Site da HE:labs
==============

http://helabs.com.br

## Running

1. Start Jekyll
```sh
$ jekyll -w serve
```

2. Start Compass
```sh
$ compass watch
```

## Editing

You can edit the SCSS files in the sass folder.
In the stylesheets folder we have just the generated files by Compass.

## Import blogposts
```sh
$ rake import:blogposts
```
