StartupDEV Workshop

Aprenda como entregar software em 2 dias

Usamos diversas técnicas para conseguir finalizar os projetos neste deadline.

Nós estamos abrindo esta oportunidade para desenvolvedores, sejam programadores ou designers, acompanharem aqui no escritório o desenvolvimento do Startup DEV.

Estamos abrindo apenas 1 (uma) vaga para que você possa vir ao nosso escritório, no Centro do RJ, nos dois dias de desenvolvimento e acompanhar todo o trabalho, com o objetivo de aprimorar seus skills em programação e design de produtos web.

É uma forma um pouco diferente de um curso. Ao invés de ensinarmos programação de software e desenvolvimento de aplicativos na teoria, nós ensinaremos na prática, enquanto estivermos desenvolvendo um projeto real que possui cliente real e um deadline apertado (ou seja, vida real).

O Startup DEV é o momento que somos mais produtivos programando e é a hora que usamos as soluções mais avançadas para resolver os problemas.

Informações Técnicas:

  • Programação em Ruby usando Rails 3
  • Usamos Mac OS com Textmate
  • Usamos banco de dados PostrgreSQL ou MongoDB
  • Desenvolvemos com Test-Driven Development (TDD)
  • Todos os projetos tem 100% de cobertura de testes
  • Fazemos testes unitários e testes de aceitação
  • Realizamos integração contínua e deploy contínuo no Heroku
  • Automatizamos várias tarefas no Heroku
  • Usamos controle de versão git com Codeplane
  • Escrevemos código para segurança de todas as aplicações
  • Design usando Photoshop + Fireworks + Illustrator
  • Front-end com HTML5 + CSS3 + jQuery

No final do segundo dia fazemos a retrospectiva do projeto em um bar, em geral na lapa. Nós gravamos a conversa da retrospectiva e bebemos cerveja e você será nosso convidado (a cerveja por nossa conta).
