Desenvolvimento Ágil

O DesenvolvimentoÁgil.com.br é um projeto Open Source feito com o apoio da ImproveIT com a HE:labs.

  • pah

    Command line interface to generate rails apps including setting up Heroku and more.

  • jumpup

    Jumpup is a gem that provides a set of tasks to automate all steps of a synchronous CI process, that is, continuous integration without a server such as TravisCI. Why? Because that's the way we like it!

  • static_content

    static_content provides a simple way to embed static content on your application that can be later edited through an admin interface.

  • acts_as_hashed

    acts_as_hashed makes it easier to create unique non-discoverable URIs and all other sorts of uuid you may need in your app.

  • favorite_it

    favorite_it helps favoriting stuff by users by providing a simple to use record, similar to acts_as_taggable.

  • sigame

    sigame is a twitter-like follow system easy enough to add to your app.
