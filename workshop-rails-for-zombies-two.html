Codeplane - Rails for Zombies 2

Vire um zumbi boladão de Rails com o curso presencial Rails for Zombies!

Um curso do Codeschool em parceria com a HE:labs, totalmente em portugues, ao vivo e a cores.

Sua cabeça vai explodir e não estamos falando de Headshots não, o Rails for Zombies 2 é uma ótima pedida para quem quer se tornar um sobrevivente e encarar as linhas de comando zumbis apenas com sua shotgun e o terminal a mão. Caia pra dentro e faça o curso Rails for Zombies 2 que vai rolar dia xx no Bees Office com os instrutores Rodrigo Pinto e Mauro George.

Level 1 - From de ground up

Como instalar Rails, criar a Rails app, migração de banco, a linha de comando, Ruby 1.9 hash syntax, Bundler, configuração do banco

Level 2 - Models still taste like chicken

Named scopes, callbacks, has_one, relationship options, using Includes, has_many :through

Level 3 - Rest in pieces

Revisiting URL helpers, forms & input helpers, understanding REST, nested resources, view partials, other view helpers

Level 4 - Asset packaging and mailing

Sending attachments in mail, asset tags, using the Asset Pipeline, CoffeeScript, creating and sending mail, SCSS, using the JavaScript manifest

Level 5 - Rendering Extremities

Custom RESTful routes, creating AJAX links, rendering custom JSON, AJAXified forms, controller rendering options, sending server JavaScript, using JSON data
