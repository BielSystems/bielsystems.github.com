Nossos Processos

Saiba como a mágica acontece.

Segunda-feira

O cliente e o time conversam sobre o projeto visão de curto e longo prazo. O cliente e o time desenham juntos como imaginam o projeto que será desenvolvido. O cliente prioriza as tarefas que serão realizadas durante a semana.

Terça e Quarta-feira

O time trabalha nas tarefas priorizadas pelo cliente. O projeto que está sendo desenvolvido fica disponivel para o cliente em um ambiente de testes. O time utiliza de testes automatizados e todas as demais práticas do XP.

Quinta-feira

O time finaliza as tarefas da iteração. No final do dia é entregue o roteiro de testes para o cliente que tem de sexta a segunda para testar o que foi desenvolvido e trazer suas observações para próxima reunião.

Sexta-feira

Desenvolvimento dos projetos internos da HE:labs. O time contribui para projetos open source. É livre o consumo de cerveja, red bull e refrigerante para o time.
